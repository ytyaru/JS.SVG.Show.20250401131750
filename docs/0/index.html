    • ローカルで保持したフォントファイルを参照する（CSS関数local()）
  1. 単色SVGアイコンをInkscape等で作成
  2. 1をfontconv等でWOFF2に変換

  4. HTMLで該当文字コードまたはHTML特殊文字を記入
